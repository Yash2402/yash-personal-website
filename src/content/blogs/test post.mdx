## Pygame Example
```python
import pygame
SCREEN = pygame.display.set_mode((512, 512))

run = True
while run:
    SCREEN.fill((0, 0, 0))
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            run = False

    pygame.draw.circle(SCREEN, (255, 255, 255), (256, 256), 20)
    pygame.display.update()
```
